# time-span

This is a backwards-incompatible fork of [local-time-duration](https://github.com/enaeher/local-time-duration), refurbished and modified to support months and years in ISO 8601-2-compatible time periods. It is a simple library that provides a `duration` class and some useful accompanying operations. It is built on and requires the [local-time](http://common-lisp.net/projects/local-time/) library.

The rest of the README and tests are incomplete.
